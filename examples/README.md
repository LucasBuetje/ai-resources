## Contributing Examples

If you've used these prompts and have good examples, consider contributing:

1. Remove any sensitive or proprietary information
2. Anonymize if necessary
3. Submit via pull request to the `examples/` directory

Please ensure examples demonstrate:
- The input provided to the custom AI
- The output generated
- Any improvements or insights gained
